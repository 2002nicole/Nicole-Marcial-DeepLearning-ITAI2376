## Project Summary

The Cookie Recipe Adaptation Agent is a single AI agent that helps beginner and intermediate bakers adapt cookie recipes. It can help with ingredient substitutions, batch size changes, dietary adjustments, and texture guidance.

## Why This Project Was Separate

This final project was submitted as its own complete repository because it includes the full working agent, notebook, requirements file, reflection, demo video, data folder, documentation, and environment setup files.

To avoid duplicating files across repositories, this ITAI 2376 portfolio repository links to the complete final project repository instead.

## Key Features

- Single-agent design
- LangChain-based tool workflow
- OpenAI-powered language model
- Retrieval-Augmented Generation using recipe and baking knowledge
- Chroma vector store
- Hugging Face sentence-transformer embeddings
- Custom recipe retrieval tool
- Custom ingredient scaling tool
- Custom substitution guidance tool
- Custom full recipe adaptation tool
- Optional Gradio interface
- Demo video included in the final project repository
